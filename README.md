# Calculator

A simple calculator built using flex & bison

## Dependencies
flex & bison

## Compile
```sh
make
```
## Usage
Start calculator using:
```sh
./calculator
```
Enter math expressions and press enter to get the results; Ctrl+c to exit.

## Features
- plus `+`
- subject `-`
- divide `/`
- multiply `*`
- absolute `|`
- parentheses `()`

## Example test expression

```
|1+-9*|-2*(2+2)|/2-5|
```
the result shall be `40`
